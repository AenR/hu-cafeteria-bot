# Hacettepe Yemekhanecisi
Hacettepe Yemekhanecisi is a Telegram bot that shares the Hacettepe Cafeteria menu on Telegram daily.

## Usage
You can join the Telegram channel from [here](https://t.me/hacettepeyemekhane)
